.. _requirements:

Requirements
=============

- python >= 3.4
- muffin >= 0.0.99
- muffin-redis >= 0.0.6

.. _installation:

Installation
=============

**Muffin-Cache** should be installed using pip: ::

    pip install git+https://github.com/drgarcia1986/muffin-cache.git
